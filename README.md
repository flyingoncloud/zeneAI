# ZeneAI Frontend

Next.js frontend application for ZeneAI IFS therapy platform.

## Features
- User authentication UI
- Chat interface with AI
- IFS therapy interface
- File upload (images, audio)
- Drawing canvas
- Session management
- Responsive design

## Setup
1. Install dependencies: `npm install`
2. Copy `.env.local` and update API_URL if needed
3. Start development server: `npm run dev`
4. Make sure backend is running on http://localhost:8080

## Environment Variables
- `NEXT_PUBLIC_API_URL` - Backend API URL (default: http://localhost:8080)

## Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server

Frontend runs on http://localhost:3000
